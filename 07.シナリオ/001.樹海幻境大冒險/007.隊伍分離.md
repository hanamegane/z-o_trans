# 隊伍分離
## シナリオ名
 - 隊伍分離
   - 

## 原文
```
赫利德:

可惡，頭好痛，大家去那裡了。 
```

```
赫利德:

對啊，梅米？海洛伊絲？巴倫？你們在那裡？
```

```
諾曼:

老闆，怎我剛才沒聽到你喊我的名字？
```

```
赫利德:

諾曼，你沒事就好了，有看到其他人嗎？
```

```
諾曼:

老闆！危險！ 
```

```
赫利德:

又是這些樹怪花怪，諾曼保護好自己，我們 要殺出去跟其他人匯合。 
```

```
諾曼:

該死的！這些樹怪都不吃我的廣法攻擊，那 就來見識一下我的利齒吧！ 
```

```
赫利德:

等等，這些花朵看起來很危險呢！ 
```
