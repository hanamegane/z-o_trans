# 不祥予感
## シナリオ名
 - 不祥予感
   - 嫌な予感

## 原文
```
赫利德:

喝！
```

```
赫利德:

終於……一都成功了。
```

```
諾曼:

快要累死了，雖然說只是幫忙輔助，但也快要掏空我的魔力了。
```

```
赫利德:

你還抱歉，說好的撐十秒，都快要半分鐘了，你才回來，我差一點就要掛在這裡了。
```

```
諾曼:

這朵紅花周邊存有大量火元素保護著，要採下並不容易啊。
```

```
赫利德:

好了，那元素花朵到手就好了，接下來先找 到梅米他們再說，我有點不祥的預威。
```

```

```
