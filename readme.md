# What's this?
これはZold:out 鍛造屋的物語(Zold:out 鍛冶屋の物語)の日本語訳を補完する目的として作成したリポジトリです。 
このリポジトリではゲーム内文章全てを完璧に訳す事を目標としていません。 
ゲーム内文章全ての翻訳ではなく、日本版にて未実装となった文章を主な対象として翻訳しています。  
  
また、翻訳に関してはすべて非公式のものです。  
このリポジトリや訳文などに関して公式へ問い合わせることはご遠慮下さい。
